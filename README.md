# PIZZERÍA ARSA

Sitio web estático desarrollado únicamente con HTML y CSS para promover la reducción del desperdicio de alimentos.

## Páginas incluidas

- login.html: inicio de sesión
- registrar.html: registro de usuario
- index.html: página de inicio
- menu.html: menú de comidas
- form_pedido.html: formulario de pedido
- pedidos.html: tabla de pedidos realizados
- ofertas.html: ofertas especiales

## Mejoras aplicadas 

- Archivo externo styles.css compartido por todas las páginas.
- Uso de Flexbox en la navegación.
- Uso de Grid Layout para organizar secciones, tarjetas, formularios y contenido.
- Animaciones y transiciones CSS en botones, tarjetas y secciones.
- Diseño responsive con Media Queries.
- Corrección de tildes y textos.
- Mejor presentación visual sin usar JavaScript ni backend.
